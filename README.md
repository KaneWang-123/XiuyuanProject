# XiuyuanProject

A personal repository for reading and annotating theoretical high-energy physics papers, primarily on **non-invertible symmetries**, **flavor physics**, and **string compactifications**.

## Contents

## Paper Sources

Papers are stored as LaTeX source (downloaded from arXiv via "Other formats → Download source").
This preserves all equations and structure for direct reading by Claude Code.

Folder naming convention:
- On arrival: `arXiv-{id}v{n}/`
- After reading: renamed to the paper title
